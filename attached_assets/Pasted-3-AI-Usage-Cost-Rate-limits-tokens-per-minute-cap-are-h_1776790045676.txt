3. AI Usage & Cost

Rate limits (tokens per minute cap) are handled in three layers: (1) each user is capped at 50 AI calls per day and 20 per minute, so no single user can drain the shared bucket; (2) if OpenAI temporarily throttles us, the system automatically retries with a short wait instead of showing an error; (3) common repeated queries (like legal research on the same topic) are cached and served instantly without hitting the API again. For growth beyond a few hundred active users, we can upgrade the OpenAI plan tier which multiplies the limits automatically.
