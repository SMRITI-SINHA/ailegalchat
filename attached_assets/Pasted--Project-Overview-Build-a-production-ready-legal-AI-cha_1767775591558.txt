
Project Overview

Build a production-ready legal AI chatbot for the Chakshi platform with intelligent document processing, multi-layer RAG reasoning, legal drafting capabilities, and cost-optimized model selection. The system must handle 20k-40k token documents  and queries at ₹0.40/query while maintaining lawyer-grade accuracy. the user can upload upto 600-800 pages of documents to ask question. also will be able to press mic icon and chat with the chabot..


Core Architecture Requirements

1. Technology Stack

Backend: Node.js with Express/Fastify
Database: PostgreSQL with pgvector for embeddings
Vector Store: Pinecone or Qdrant for scalable semantic search
File Storage: AWS S3 or Cloudflare R2
OCR: Tesseract.js or cloud OCR (Google Vision API fallback)
PDF Processing: pdf-parse, pdf.js
API Integration: OpenAI SDK with custom routing logic
Frontend Integration: REST API with WebSocket support for streaming
Web Scraping: Playwright (for JS-heavy sites),
Job Queue: Bull/BullMQ for background processing
Admin UI: React Admin or Refine for rapid dashboard development
File Processing: multer (upload), archiver/unzipper (archives)
Webhooks: Ngrok for local dev, production webhooks for Drive/cloud

2. Document Intake & Processing System

Requirements:


Accept multiple formats: PDF, scanned images, Word docs, case bundles
OCR processing for scanned documents with 95%+ accuracy
Text extraction handling 20k-40k token documents
Intelligent chunking with legal context awareness (maintain section integrity)
Metadata extraction: case numbers, dates, parties, courts, citations
Cost target: ₹1 per document using nano/mini models

3. Multi-Layer RAG & Legal Reasoning Engine

Layer 1 - Fast Understanding (GPT-4o-mini):


Vector similarity search (top-k=10)
Hybrid search: semantic + keyword matching
Initial answer generation with citations
Confidence scoring
Cost: ~₹0.15 per query
Layer 2 - Error Checker (GPT-4.1):
Contradiction detection against source documents
Statute compliance verification
Hallucination detection using grounded verification
Citation accuracy check
Triggered when: confidence < 0.8 OR legal complexity detected
Cost: ~₹0.20 per verification
Layer 3 - Deep Reasoning (o1-pro):
Complex legal interpretation
Multi-jurisdiction analysis
Ambiguous precedent resolution
Only triggered for: constitutional matters, conflicting precedents, novel legal issues
Cost: ~₹2.00 per deep analysis (used <5% of queries)
